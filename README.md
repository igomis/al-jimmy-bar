# Al Jimmy Bar

Repositori de treball per escriure **Al Jimmy Bar**, una novel·la de gènere adolescent.

## Estructura

- `manuscrit/`: capítols i índex del llibre.
- `notes/`: bíblia de la novel·la, personatges, escaleta, món narratiu i idees pendents.
- `recursos/`: imatges, referències o documents auxiliars.
- `docs/`: criteris d’estil, planificació i materials de suport.

## Flux recomanat

1. Escriure cada capítol en un fitxer Markdown dins de `manuscrit/`.
2. Mantindre les decisions narratives en `notes/`.
3. Fer commits freqüents amb canvis menuts.
4. Usar branques si vols provar versions alternatives d’un capítol.

## Estat

Projecte inicialitzat. Document de referència principal: `notes/biblia-novella.md`.
