# Guia d’estil

- Text visible i narració en valencià, llevat que un personatge parle d’una altra manera per motius narratius.
- Frases clares i escenes amb acció concreta.
- Evitar explicar massa el que el lector pot deduir per diàlegs i gestos.
- Mantindre una veu coherent amb personatges adolescents sense forçar argot.
- Cada capítol hauria d’avançar almenys una d’aquestes coses: conflicte, relació, misteri o canvi intern.
